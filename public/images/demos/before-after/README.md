# Före/Efter Slider - Exempelbilder

Detta är demo-bilder för Före/Efter Slider showcase-funktionen.

## Bildspecifikationer

- **Format:** JPG
- **Dimensioner:** 1200x800px (3:2 aspect ratio)
- **Max filstorlek:** 300KB per bild
- **Kvalitet:** Hög kvalitet men komprimerad för webben
- **Viktigt:** FÖRE och EFTER måste vara samma vinkel/komposition för bästa effekt

## Bildpar

### 1. Renovation (Hemrenovering)
- `renovation/before.jpg` - Före renovering (kök/badrum)
- `renovation/after.jpg` - Efter renovering (samma rum)

### 2. Web Design (Webbdesign)
- `web-design/before.jpg` - Gammal hemsida (screenshot)
- `web-design/after.jpg` - Ny hemsida (screenshot, samma sida)

### 3. Photo Edit (Fotoredigering)
- `photo-edit/before.jpg` - Obehandlad bild
- `photo-edit/after.jpg` - Redigerad bild (samma foto)

### 4. Landscape (Landskapsdesign)
- `landscape/before.jpg` - Före trädgårdsarbete
- `landscape/after.jpg` - Efter trädgårdsarbete (samma plats)
